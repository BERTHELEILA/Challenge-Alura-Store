**Alura Store Challenge**

**Descripción:**
Este proyecto corresponde al desafío de análisis de datos de Alura Store, realizado en Google Colab. El objetivo principal es explorar y analizar la información de ventas, productos y tiendas para obtener insights que ayuden a mejorar la gestión comercial y la toma de decisiones.
El análisis incluye ingresos por tienda, costos de envío, clasificación promedio y productos vendidos por categoría.

**Tecnologías Utilizadas:**
Python 3
Google Colab como entorno de desarrollo
Pandas para manipulación de datos
Matplotlib para visualización de gráficos

**Datos:**
Los datos provienen de un conjunto proporcionado en el desafío de Alura Store y contienen información sobre:
- Ingresos totales por tienda
- Costo de envío promedio por tienda
- Clasificación promedio de tiendas
- Total de productos vendidos por categoría

**Análisis Realizado:**
Ingresos Totales por Tienda: Se identifican las tiendas con mayor facturación mediante gráficos de barras.
Costo de Envío Promedio por Tienda: Comparación de los costos de envío promedio entre tiendas.
Clasificación Promedio por Tienda: Evaluación de la satisfacción promedio de los clientes por tienda.
Productos Vendidos por Categoría: Determinación de las categorías más vendidas para identificar tendencias.

**Cómo Usar:**
1. Clonar este repositorio: git clone https://github.com/BERTHELEILA/Challenge-Alura-Store.git
2. Abrir el notebook AluraStoreLatam.ipynb en Google Colab
3. Ejecutar las celdas para reproducir los análisis y gráficos.
4. Explorar los resultados y visualizaciones generadas para obtener insights.
